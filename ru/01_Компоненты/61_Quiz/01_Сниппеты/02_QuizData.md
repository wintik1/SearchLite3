# QuizData

Сниппет предназначен для вывода заполненных данных с формы.
В основном нужен для вывода данных в письме.

## Параметры:

- **id** - id квиза.
- **param** - параметры формы для получения ответа в виде массива.

## Пример:

	{'!QuizData' | snippet: [
	    'id' => 2,
	    'param' => [
	        'earch' => 'ball', 
	    ],
	] | print}

Ответ:
	
	Array
	(
	    [0] => Array
	        (
	            [question] => Какой формы Земля?
	            [answer] => Шар
	            [name] => earch
	        )
	)
